### To get started with this project, follow these steps:

1. **Create the repository:**

   - Open **VS Code**.
   - From the top menu, go to **View > Source Control**.
   - Click on **Initialize Repository** or **Publish to GitHub** in the Source Control pane.
   - Select a directory on your machine where you want to store the project.
   - This will initialise a Git repository in the selected folder.
   - If you published to GitHub, you will be able to access the repository anywhere you can login to a computer. This is highly recommended.

2. **Create your project files:**

   - Create and edit the files as and when you need to add something.
   - VS Code will automatically detect the changes and show them in the Source Control pane.
   - Before the next step, modify the contents of index.html

3. **Commit your changes:**

   - In the Source Control pane, type a commit message in the input box at the top (e.g., "Initial commit").
   - Click on the **✔️ Commit** button to save your changes.

4. **Push your changes (if using a remote repository):**
   - If you're using a remote repository (e.g., GitHub or another Git server), you can push your local repository.
     - Open the **Terminal** in VS Code (`Ctrl+` `) and run the command to add the remote repository URL:
       ```
       git remote add origin <REMOTE_URL>
       ```
     - Push your local changes to the remote repository:
       ```
       git push -u origin main
       ```
