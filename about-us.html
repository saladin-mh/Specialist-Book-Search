About SMH Library

SMH Library is a modern, accessible online platform designed to help readers find, rate, and save their favourite books. Developed with inclusive design principles, the website supports both discovery and engagement through interactive search filters, wishlisting features, favourites, and a virtual book club.

This application reflects best practices in front-end development, combining dynamic JavaScript functionality, semantic HTML, and responsive CSS to create an intuitive and inclusive reading experience for all users.
